# Day 36

## 目标：devTools

- openging devTools
  - `cmd + option + i` Open whatever panel you used lastTab
  - `cmd + option + j` Open Console
  - `cmd + shift + c` Open Elements(鼠标变成查看元素)
- 打开 console 面板
  - `ctrl + ` `
- 清空 console 面板
  - `cmd + k`
- 搜索文件(打开 devTools 后)
  - `cmd + p`
  - 搜索命令`cmd + shift + p`
- 切换面板
  - `cmd + [` &nbsp;or&nbsp; `cmd + ]`
- 在 sources panel 中显示隐藏底部面板(关闭console页面)
  - `ESC`
