### 目标：掌握文本对象
- 文本对象： 文本是结构化的，可以快速选择   范围
- 内部：i   外部：a
- operator + (内部/外部) + 文本对象
- w 一个单词
- （ 或 ) 或 b => 一对() 
-  [ 或 ]  => 一对[] 
-  { 或 } 或 B => 一对{}
-  < 或 > => 一对<>
-  t  => xml标签
- 技巧： 删除一个参数： daa
-       修改一个参数： cia
- 常用： vib 选择(括号)内的东西   viw 选中当前单词  ciw / diw 删除当前单词
-       yaw 复制整个单词
